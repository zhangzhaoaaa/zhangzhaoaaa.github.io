CSS三角形探秘

发表于 2017-03-12   |   分类于 CSS

今天我们来分析一下使用CSS如何绘制三角形的例子
如果我们认为CSS其实只是一个装饰器的话，就会有这样的疑惑，CSS怎么可能画一个三角形呢，不可能吧？
哎，它还真的可以绘制

我们先明确几个知识点
border-width和border-color的属性值的顺序是：top-right-bottom-left

总结

  1. 设置div的宽高为0
  2. 设置border-width，也就是三角形的高
  3. 设置border-style，设置为solid
  4. 之后就是隐藏或者不设置某些方向的三角形
