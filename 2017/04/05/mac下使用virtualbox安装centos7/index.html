我的虚拟主机名为centos

设置Host-Only

通过端口转发在NAT模式下实现主机访问虚拟机，但是每个端口都需要设置很麻烦。另一种办法就是设置一个Host-Only网络，为虚拟机增加一个Host-Only的网络适配器，实现主机与虚拟机之间的互通。
首先，需要创建一个Host-Only网络，路径VirtualBox->Preferences->Network->Host-Only Networks点击新增，会创建vboxnet0。

关闭虚拟机，设置虚拟机Settings->Network->Adapter2->Enable Network Adapter选择Host-Only Adapter，Name选择刚创建的vboxnet0，点完成，重启虚拟机。
虚拟机中使用ip addr可以看到多了一个网卡enp0s8的IP地址192.168.56.101
//使用ssh通过该IP访问虚拟机
ssh root@192.168.56.101 //OK，成功
