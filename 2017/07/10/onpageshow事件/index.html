onpageshow事件

发表于 2017-07-10   |   分类于 JS基础

onpageshow

定义和用法

onpageshow事件在用户浏览网页时触发。
onpageshow事件类似于onload事件，onload事件在页面第一次加载时触发，onpageshow事件在每次加载页面时触发，即onload事件在页面从浏览器缓存中读取时不触发。
为了查看页面是直接从服务器上载入还是从缓存中读取，可以使用PageTransitionEvent对象的persisted属性来判断。如果页面从浏览器的缓存中读取该属性返回true，否则返回false。

语法

  • HTML
  • JavaScript
    object.onpageshow = function(){myScript};
  • addEventListener
    object.addEventListener(“pageshow”, myScript);
    技术细节
  1. 不支持冒泡
  2. 不可以取消
  3. 事件类型为PageTransitionEvent
  4. 支持的HTML标签是

用例

当页面加载的时候，重新加载页面

window.onpageshow = function(event) {
    if (event.persisted) {
        window.location.reload();
    }
}
