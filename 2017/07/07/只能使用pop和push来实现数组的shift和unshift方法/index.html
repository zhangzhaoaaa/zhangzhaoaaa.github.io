只能使用pop和push来实现数组的shift和unshift方法

发表于 2017-07-07   |   分类于 算法设计

题目

除了pop和push之外不能使用数组其他任何方法，也不能新增数组，如给定数组是nums，里面有n个数字，来实现数组的shift和unshift方法

分析

我本来想利用扩大nums数组长度来搞，但是搞了之后发现没办法回收扩出去的空间，所以还是得利用它自己的空间处理，因为shift和unshift处理的是数组的头部，pop和push处理的是数组的尾部，所以只能先将数组反转然后调用对应方法，最后再反转回来
