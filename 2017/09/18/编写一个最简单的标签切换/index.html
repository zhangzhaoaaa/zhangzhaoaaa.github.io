编写一个最简单的标签切换

发表于 2017-09-18   |   分类于 JS应用

标签组件是我们工作中常用的组件之一，今天我们分析一下如何来写一个最简单的标签切换组件

使用技术

  • 原生JS
  • jQuery
  • CSS:flexbox

分析

首先我们知道一个标签切换有标签头(navigator)+标签体(content)

所以我们来先把结构写出来

html结构

一般来说标签头使用ul>li>a来表示，当然你也可以使用div，我们这里使用一个通常的做法

标签体就是div了，有几个标签头就对应几个标签体

<div>
    <div>
        <ul>
            <li><a>Pictures</a></li>
            <li><a>Music</a></li>
            <li><a>Videos</a></li>
            <li><a>Documents</a></li>
        </ul>
    </div>
    <div>
        <div>123</div>
        <div>456</div>
        <div>789</div>
        <div>abc</div>
    </div>
</div>

样式和交互

结构写出来之后，我们就需要考虑一下tabs的样式和交互状态了

一般tabs都有一个默认激活标签，其他标签默认是隐藏状态。因此在样式上，默认标签需要一个激活状态，其他标签是默认状态。先说明，我们这里处理的tabs是横向的标签。然后，交互上，点击某个标签头，当前标签头处于激活状态，标签体显示出来，其他的标签头变成默认状态，标签体变为隐藏状态。在鼠标滑过的时候，默认标签头样式有个变化。

ok，我们来看下样式

添加样式后的html结构

<div class="container">
    <div class="tabs">
        <ul>
            <li class="is-active tab"><a>Pictures</a></li>
            <li class="tab"><a>Music</a></li>
            <li class="tab"><a>Videos</a></li>
            <li class="tab"><a>Documents</a></li>
        </ul>
    </div>
    <div>
        <div class="tabpane">123</div>
        <div class="tabpane">456</div>
        <div class="tabpane">789</div>
        <div class="tabpane">abc</div>
    </div>
</div>

交互代码如下
jQuery

$(function () {
    $('.tab').click(function(){  // 标签头
        var index = $(this).index(); // 当前标签头的序号
        $(this).addClass("is-active").siblings().removeClass('is-active'); // 将当前点击的标签头加上激活状态，兄弟标签头去除激活状态
        $('.tabs-body .tabpane').eq(index).show(200).siblings().hide(); // 将对应的标签体显示，其他的兄弟标签体隐藏
    });
    $('.tabs .tab:eq(0)').click(); // 点击第一个标签头
});

原生JS

var navs = document.querySelectorAll('.tab');  // 标签头
var cons = document.querySelectorAll('.tabpane'); // 标签体

for (var  i = 0; i < navs.length; i++) {
    navs[i].index = i;  // 记录每个标签头的序号
    navs[i].onclick = function () {
        for(var n = 0; n < cons.length; n++) { // 将所有标签体和标签头样式reset
            cons[n].style.display = "none";
            navs[n].className = "";
        }
        cons[this.index].style.display = "block" // 当前标签体显示
        navs[this.index].className = "is-active"; // 当前标签头激活
    }
}
navs[0].click(); // 触发第一个标签头点击事件

结论

以上都是最原始，最简单粗暴的一种tab切换的实现方法，其他比如使用框架或者面向对象的方式实现tabs，都可以围绕以上的代码进行改造
