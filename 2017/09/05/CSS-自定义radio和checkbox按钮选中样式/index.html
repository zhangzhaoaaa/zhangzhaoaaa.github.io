CSS 自定义radio和checkbox按钮选中样式

发表于 2017-09-05   |   分类于 CSS

我们在开发中，不可避免要使用单选和多选按钮，但是原生样式相对来说不是那么美观，所以我们需要重新定义它们的样式以及选中的样式。

我们该如何着手做呢，首先我们需要了解几个概念

基本概念

  1. :after(或者:before)伪类
  2. :checked伪类
  3. label点击的交互

首先对于label标签来说，它的特性就是会和一个控件进行关联，比如

点击文字，label包裹radio或者一个label的for对应了radio的id效果都是一样的，以上就会在两个radio选中之间切换

有了这个特性，我们就可以想象，哎，我把radio button隐藏了，然后在label中放一个span或者其他HTML元素，定义一个选中样式，一个未选中样式，这不就达到我想要的目的了么

比如我们就使用span来定义这个自定义样式的元素（也可以选择其他HTML元素，最好inline的元素）
选中样式好定义，直接在span上定义样式就行了，但是选中的样式怎么办呢

幸好我们CSS中有个:checked的伪类

:checked CSS 伪类选择器表示任何处于选中状态的radio(), checkbox () 或(“select”) 元素中的option HTML元素(“option”)) 。用户通过点击元素或选择其他的值，可以改变该元素的 :checked 状态，并:checked属性赋给一个新的对象(例如其他的option值)。

通过:checked，我们可以给选中的radio定义样式

我们不想通过js来改变span这一个元素的样式，这时候就需要一个新的元素来和span的样式进行叠加处理，:after（或:before）能够很方便的满足这个需求，这样，我们就会有一个css选中选择器，就是:checked下的span:after负责选中状态
