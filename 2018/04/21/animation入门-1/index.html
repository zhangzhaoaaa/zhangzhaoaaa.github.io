animation入门

发表于 2018-04-21   |   分类于 CSS

CSS中属性animation可以被用在很多CSS属性上，比如color、background-color、height或者width。每一个动画需要使用@keyframes来定义，然后使用animation属性来调用，比如：

每一个@keyframes的规则定义了在动画期间在特定时刻什么应该发生。比如，0%是动画的开始，100%是结束。这些关键帧(keyframes)可以被短的animation属性控制，也可以为了更多的控制这些关键帧使用animation的8个子属性来控制。

子属性

  • animation-name:定义了@keyframes规则的名称
  • animation-duration: 完成一个周期的动画花费的时间长度
  • animation-timing-function: 建立预置的加速度曲线，比如：ease 或者linear
  • animation-delay: 元素加载完成后与动画序列开始时之间的间隔时间
  • animation-direction: 设置动画的方向。在每一次循环会重置
  • animation-iteration-count: 动画执行的次数
  • animation-fill-mode: 设置哪个值应用在动画的前面还是后面
    比如，你要设置动画的最后一个状态是停留在屏幕上，或者你可以将其设置为在动画开始之前切换回之前的状态
  • animation-play-state: 暂停/播放 动画

这些子属性可以这样使用：

以下是一些子属性可使用的值
属性 | 值
—–|——
animation-timing-function | ease,ease-out,ease-in,ease-in-out,linear,cubic-bezier(x1,y1,x2,y2)
animation-duration | Xs or Xms (单位为秒/毫秒)
animation-delay | Xs or Xms (单位为秒/毫秒)
animation-iteration-count | X
animation-fill-mode | forwards,backwards,both,none
animation-direction | normal,alernate
animation-play-state | paused,running

性能

大多数属性的动画是有性能方面的考虑，所以我们在采用任何一个属性做动画的时候都应该谨慎。但是，有几个组合使用动画是安全的：

  • transform: translate()
  • transform: scale()
  • transform: rotate()
  • opacity
